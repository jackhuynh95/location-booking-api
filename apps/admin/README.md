# Admin App

Future home of an optional admin client.

Possible responsibilities after approval:

- Manage locations.
- View the location tree.
- Display the original assignment seed data through backend API calls.
- Create and inspect bookings.
- Exercise API validation scenarios.
- Build to static assets that the NestJS server can serve in the final Docker runtime.

This app is a placeholder only. Do not scaffold a frontend framework until the user confirms the admin scope.
